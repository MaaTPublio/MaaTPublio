## Projeto em Destaque

### [Implementação de SOC Corporativo (Wazuh + pfSense)](https://github.com/MaaTPublio/implementacao-soc-corporativo)
> *Case de implementação real em ambiente de produção.*

Construção de uma arquitetura de segurança defensiva integrada, protegendo ativos corporativos contra ameaças reais.
- **SIEM:** Centralização de logs e correlação de eventos com Wazuh.
- **Firewall/IDS:** Segurança de borda com pfSense, Suricata e pfBlockerNG.
- **Endpoint:** Telemetria avançada via Sysmon e resposta ativa a incidentes (Active Response).

[Ver Repositório Completo →](https://github.com/MaaTPublio/implementacao-soc-corporativo)
